# SkillOS

The developer skill verification platform. Developers prove their skills through coding challenges. Recruiters find and contact verified talent.

See README below for full docs — this is a self-contained Python + React monorepo.

## Quick Start

```bash
# Backend
cd backend && pip install -r requirements.txt
python -m skillos.main

# Frontend  
cd frontend && npm install && npm run dev
```

Server: http://localhost:8000 | Frontend: http://localhost:5173

See DOCS.md for full API reference, deployment guide, and environment variables.
